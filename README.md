# IOT-PBL
room sensor, made for pressure, tmp and hmd, uses a display and led for feedback. 
Core Requirements (All Teams Must Implement)
•	Sensor Integration: Continuously measure temperature, humidity, and pressure using BME280
•	Local Display: Show readings on OLED in a clear, user-friendly format with regular updates
•	Web Interface: Provide a web dashboard accessible via local network for remote monitoring
•	Visual Alerts: Implement RGB LED warnings for critical conditions (too hot, too humid, poor air quality)

Enhanced Features 

•	Data logging with timestamped records
•	Low-power operation with sleep modes
